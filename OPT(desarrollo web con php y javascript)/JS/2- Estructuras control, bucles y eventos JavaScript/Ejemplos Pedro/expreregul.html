Expresiones regulares

Las expresiones regulares por permite comprobar si una cadena cumple con un patron en específico

Podemos definir los rangos escribiendolo entre [], [0-9], [A-Z]...

Tenemos caracteres que implican condiciones en especificas como ^, que indica que la cadena tiene que empezar con la cadena que lo sigue

$, la cadena debe terminar con la cadena descrita a continuación

Los cuantificadores mas comunes son +, *, y ?, que significan respectivamentes 1 o mas, 0 o mas y si el elemento anterior coincide con la cadena
